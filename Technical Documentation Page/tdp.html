Brief History of PHP

PHP (Hypertext Preprocessor) is a widely-used open-source scripting language primarily designed for web development. It was created in 1994 by Rasmus Lerdorf, a Danish-Canadian programmer, as a set of Common Gateway Interface (CGI) scripts written in C. Initially, Lerdorf developed PHP to track visits to his online resume, naming it "Personal Home Page Tools" (PHP Tools).

In 1995, PHP/FI (Form Interpreter) was released as the first version, with basic functionality for web development, allowing users to create simple dynamic websites. Due to its ease of use and integration with HTML, PHP quickly gained popularity. By 1997, PHP had undergone significant improvements, and two Israeli developers, Zeev Suraski and Andi Gutmans, rewrote its core, leading to the release of PHP 3 in 1998. This version rebranded PHP as "PHP: Hypertext Preprocessor" and introduced support for a broader range of databases and protocols, which greatly expanded its functionality.

PHP 4, released in 2000, introduced the Zend Engine, which dramatically improved performance. Subsequent versions continued to enhance security, performance, and object-oriented programming features. PHP 5, released in 2004, further strengthened object-oriented programming, and PHP 7, released in 2015, brought significant speed improvements and reduced memory usage.

PHP Syntax for Beginners

To use php, we'll need to use this:

<?php ?>

Few Key Notes:

  • You can actually put no semicolon on the last line in a block of PHP. But it’s recommended to always put a semicolon
  • If your file is pure php code, you don't need to put the closing tag "?>"

To display some simple html:

<?php if(true){
  <p>Some HTML text...</p>
?>

Comments:

  • User "//" for single Line
  • Use "/*-- --*/" for multiple Lines
<?php
  //This right here is a comment
  <p>Some HTML text...</p>
?>
PHP Variable and Data Type

To declare a variable:

  • By using "$" + the desired variable name

Example:

<?php
  $name = "Someone"
  echo $name;
?>

To declare a data type:

  • Scalar types

Note: The “string”, “int”, “float”, “bool” is just a variable name. in PHP, you generally do not need to declare the data type of a variable explicitly.

<?php
  $string = "Someone"
  $int = 1234567
  $float = 2.567
  $bool = true
?>
  • Array types

You could do it in two ways:

<?php
  $variableName = array('a', 'b', 'c'); //For PHP version 5.4 or older
  $variableName2 = ['a', 'b', 'c']; //For newer version
?>
  • Object type

Object - Something we create based of a “class”;

<?php
  $object = new Car();
?>
Built-In Superglobal Variables in PHP

Superglobal Variables always starts with “$_” + capitalized word.

  • $_SERVER["DOCUMENT_ROOT"]; - Used to get the information about the root path of a particular website
  • $_SERVER[“PHP_SELF”]; - Used to get the root folder and the file name of a file.
  • $_SERVER[“SERVER_NAME”]; - To get the name of the server
  • $_SERVER[“REQUEST_METHOD”]; - It’ll get what request_method is used. Request methods are like the things in the node.js, the get, post, put, delete, etc… method.
  • $_GET - To get something from array or database.
  • $_POST - To add something in the database.
  • $_REQUEST - not be confused by “$_SERVER[“REQUEST_METHOD”]”... also work to output/input from or in the database. Meaning it works in many ways, so it’s much recommended to just use the specifics like $_GET or $_POST, etc…
  • $_FILES - To get all the information about the files that’s going to be submitted/uploaded
  • $_COOKIE - Small files that the server embeds on the user’s computer
  • $_SESSION - used to store and manage session data across multiple pages of a web application. Sessions in PHP allow you to persist data (like user information) across different pages during a user's visit.
